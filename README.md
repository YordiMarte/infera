# Infera.JS

AI Inference Gateway designed to proxy LLM requests with
rate limiting, caching, observability and token-based billing.

## Architecture
Client → Gateway API → Inference Workers → External LLMs

## Tech Stack
- NestJS
- Go
- Redis
- PostgreSQL
- Docker
- Kubernetes (EKS)
- Terraform
